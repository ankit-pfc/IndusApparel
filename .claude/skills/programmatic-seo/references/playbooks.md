# Programmatic SEO Playbooks for INDUS Apparels

Detailed implementation guides for each pSEO template type, adapted for D2C denim e-commerce.

## Playbook 1: Product Variation Pages (Fit x Wash Matrix)

**Pattern**: `/jeans/[fit-slug]/[wash-slug]`
**Total pages**: 6 fits x 7 washes = 42 pages
**Search intent**: Buyers who know what style/color they want

### URL Examples
```
/jeans/slim-taper/rinse-indigo
/jeans/slim-taper/dark-rinse
/jeans/slim-taper/mid-indigo-eco
/jeans/athletic-taper/rinse-indigo
/jeans/athletic-taper/storm-grey
/jeans/selvedge-slim/copper-rigid
/jeans/carpenter/ecru-undyed
... (42 total)
```

### Template Structure
```
Title: [Fit Name] in [Wash Name] — Rs.[Price] | INDUS Apparels
Meta: Shop the [Fit Name] in [Wash Name]. [Fit desc]. [Fabric weight] [fabric type]. Rs.[Price] with free shipping across India.

H1: [Fit Name] — [Wash Name]

[Hero product image + color swatch]

## About the [Fit Name]
[fit.desc expanded to 100-150 words — who this fit is for, body types, occasions]

## [Wash Name] Details
[Wash description: color depth, fade characteristics, care level, how it ages]

## Fabric & Construction
- Weight: [sku.fabric weight]
- Type: [Core Stretch / Rigid / Selvedge]
- Stretch: [Yes/No + percentage if applicable]
- [Selvedge-specific details if applicable]

## Size Guide
[Size chart table specific to this fit — waist, hip, thigh, inseam, hem]

## How to Style
[3 outfit ideas specific to this fit + wash combination]
- Office look: [fit+wash] + white button-down + leather loafers
- Weekend look: [fit+wash] + graphic tee + white sneakers
- Evening look: [fit+wash] + dark polo + Chelsea boots

## Care Instructions
[Wash-specific care: wash temp, dry method, fade preservation tips]

## You Might Also Like
[3-4 related products: same fit/different wash, same wash/different fit]

[Add to Cart CTA with size selector]
```

### Unique Content Per Page
Each of the 42 pages needs genuinely different content in these sections:
- **Styling**: Slim Taper in Overdyed Black styles differently than Carpenter in Ecru
- **Care**: Raw/Rigid denim needs different care than pre-washed stretch
- **Who it's for**: Athletic Taper in Dark Rinse = gym-goer who needs office jeans vs. Selvedge Slim in Copper Rigid = denim enthusiast

## Internal Linking Strategy

### Link Density Targets
- Product variation pages: 5-8 internal links
- Body type guides: 8-12 internal links (more editorial, more cross-references)
- Style guides: 6-10 internal links
- Comparison pages: 6-8 internal links
- Care pages: 4-6 internal links
- "Best X" pages: 8-10 internal links

### Contextual Link Placement
Links should appear naturally within content, not just in "Related" sidebars:
- "If you have muscular thighs, the [Athletic Taper](/jeans/athletic-taper/rinse-indigo) is built for you."
- "Learn [how to care for raw denim](/care/raw-denim-break-in-guide) to make your Selvedge Slim last."
- "Not sure between these two? [Compare Slim Taper vs Athletic Taper](/compare/slim-taper-vs-athletic-taper)."

### Anchor Text Guidelines
- Use descriptive anchor text, not "click here"
- Vary anchor text — don't use the same text for every link to a page
- Include product names and fit names in anchors naturally

## Choosing Your First Playbook

### Start with Product Variation Pages if:
- You have product photography for all fit+wash combos
- You want to capture bottom-funnel commercial intent
- Your site needs more product pages for authority

### Start with Body Type Guides if:
- You want to capture top-funnel informational traffic
- You need content that differentiates from Levi's/Lee
- You want to build topical authority in "jeans for [body type]"

### Start with Style Guides if:
- You have India-specific occasion content (kurta combos, wedding, monsoon)
- You want to capture traffic from underserved search queries
- You have styling photography/lookbooks

### Combine Playbooks
Layer multiple page types for maximum coverage:
- Body type guide ("Best jeans for athletic build") → links to product page ("/jeans/athletic-taper/dark-rinse") → links to care guide ("/care/how-to-wash-stretch-denim") → links to comparison ("/compare/athletic-taper-vs-slim-taper")
- This creates a content loop that keeps users on site and covers the full purchase journey.
